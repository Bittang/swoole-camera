swoole-camera
=====
Web摄像头，线上体验地址：

* 摄像头：<http://182.254.148.72:9503/camera>
* 查看视频：<http://182.254.148.72:9503/>


安装swoole扩展
----
```shell
pecl install swoole
```
运行程序
----
```shell
php server.php
```

打开浏览器
----
> 目前仅支持Chrome浏览器

* 摄像头页面：<http://127.0.0.1:9501/camera>，打开页面后点击允许访问摄像头
* 查看视频内容：<http://127.0.0.1:9501/>
